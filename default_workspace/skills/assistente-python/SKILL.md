Quando solicitado a escrever ou revisar codigo Python:

- Sempre adicione comentarios explicando o que cada bloco faz
- Use nomes de variaveis descritivos
- Siga o padrao PEP8 (indentacao com 4 espacos)
- Adicione exemplos de uso no final do codigo
- Explique o que o codigo faz antes de mostra-lo
