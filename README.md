\# 🐱 Kitty AI



Assistente pessoal com IA construída do zero em Python.



\## Sobre o projeto



Kitty é uma assistente inteligente que pode ler e escrever arquivos,

executar comandos e conversar em português. Construída com base no

projeto \[build-your-own-openclaw](https://github.com/czl9707/build-your-own-openclaw).



\## Funcionalidades



\- 💬 Conversa em português

\- 📂 Leitura e escrita de arquivos

\- 💻 Execução de comandos no terminal

\- 🔌 Compatível com OpenAI, Claude e Gemini



\## Como rodar



1\. Clone o repositório

2\. Copie o arquivo de configuração:

&#x20;  cp default\_workspace/config.example.yaml default\_workspace/config.user.yaml

3\. Edite o config.user.yaml com sua API key

4\. Rode o bot:

&#x20;  uv run my-bot chat
